# Finance Tracker

A simple console-based application for tracking personal expenses by category. Written in Java.

## Features
- Add expenses with a category and amount (e.g., `food 500`).
- If the category already exists, the amount is added to the total.
- View all categories and the total spent in each.
- Exit the program by typing `exit`.

## Usage
- When prompted, enter a category name (e.g., `food`, `transport`, `entertainment`).
- Then enter the amount (an integer).
- After each addition, the program displays all current categories and totals.
- To quit, type `exit` at the category prompt.
